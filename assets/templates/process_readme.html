¿Qué hace esta pantalla?

Esta pantalla se utiliza para seguir la ejecución del proceso. El sistema muestra mensajes de avance en el estado general y actualiza cada fila con su resultado individual, incluyendo estado, ticket generado y errores cuando corresponda.

Muestra el resultado individual de cada fila procesada.

Botón Volver

Permite regresar a la pantalla anterior cuando no existe un proceso en curso.

Recomendaciones importantes

Durante la ejecución

  • Espera a que el navegador abra y cargue completamente.
  • No interactúes manualmente con la ventana del navegador mientras el proceso esté corriendo.
  • No cierres la aplicación ni el navegador durante la ejecución.

Sobre el seguimiento

  • Revisa el estado general para saber en qué etapa va el proceso.
  • Usa el detalle por fila para identificar tickets correctos o filas con error.
  • Si una fila presenta error, revisa la información de la planilla antes de reintentar.

Antes de volver

  • No intentes volver mientras exista una automatización en curso.
  • Espera a que el proceso termine antes de salir de esta pantalla.
  • Si el sistema informa un error, revisa el mensaje mostrado antes de continuar.
Recomendación: esta pantalla está pensada para seguimiento. Una vez iniciado el proceso, conviene observar el estado general y el detalle por fila hasta que la ejecución finalice.
