¿Qué hace esta pantalla?

Esta pantalla permite administrar la configuración base usada por el sistema antes de iniciar la carga de tickets. Aquí se puede definir el usuario reportante, el grupo de trabajo, restaurar los valores por defecto, guardar cambios y eliminar la sesión guardada para volver a iniciar sesión con otra cuenta.

Usuario Reportante

Define el usuario que será utilizado como reportante en la creación del ticket.

Grupo de Trabajo

Define el grupo responsable que será utilizado durante la carga.

Guardar

Permite guardar los cambios realizados en la configuración.

Restaurar

Permite volver a los valores por defecto definidos por el sistema.

Borrar Sesión

Permite eliminar la sesión guardada para volver a iniciar sesión con otra cuenta.

Volver

Permite regresar a la pantalla anterior sin iniciar la automatización desde aquí.

Recomendaciones importantes

Antes de guardar

  • Verifica que el usuario reportante sea el correcto.
  • Revisa que el grupo de trabajo corresponda al que debe utilizarse en la carga.
  • Guarda los cambios antes de volver a la pantalla principal.

Sobre restaurar

  • Restaurar elimina la configuración personalizada actual y vuelve a los valores por defecto.
  • Después de restaurar, revisa los campos antes de continuar.
  • Si necesitas una configuración específica, guarda nuevamente los datos correctos.

Sobre borrar sesión

  • Utiliza esta opción cuando necesites cargar tickets con otra cuenta.
  • Al borrar la sesión guardada, la próxima vez deberás iniciar sesión nuevamente.
  • No es necesario usar esta opción si seguirás trabajando con la misma cuenta.
Recomendación: utiliza esta pantalla para dejar revisados los parámetros antes de iniciar la carga. Si necesitas cambiar de cuenta, elimina la sesión guardada antes de volver al proceso.
