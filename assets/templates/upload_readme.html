¿Qué hace este frame?

Este frame funciona como punto de inicio. Su propósito es permitir al usuario escoger la planilla de ticket, validar y luego preparar los tickets pendientes para cargar automaticamente en la plataforma. Permite descargar la plantilla de excel usada para registrar las actividades realizadas durante el dia.

Cómo usar la aplicación

1

Seleccionar archivo

El usuario carga el Excel desde la pantalla principal.

2

Validar datos

Se revisa el formato, encabezados y columnas obligatorias.

3

Automatizar tickets

La aplicación filtra los ticket pendientes, y los crea en la plataforma.

4

Guardar resultados

Se actualiza el Excel hasta que se finalice el proceso.

Descripción de cada apartado

Cargar Planilla

En este apartado se selecciona la planilla Excel que contiene los datos a procesar. La aplicación utilizará esta información para realizar la carga de tickets.

Configuración

En este apartado se revisan los parámetros básicos de la carga. Aquí se puede verificar el usuario reportante y el grupo de trabajo que serán utilizados por el sistema.

Enviar

Este apartado permite iniciar la ejecución del proceso. Una vez comenzada la carga, el sistema mostrará el avance y el resultado de cada registro procesado.

Ícono de Excel

Este ícono permite descargar la plantilla base del sistema. La plantilla debe utilizarse como formato de trabajo para registrar correctamente la información antes de la carga.

Requisitos de la planilla Excel

La planilla debe respetar una estructura mínima para que el sistema pueda leerla correctamente y detectar los tickets pendientes.

Encabezados obligatorios

FECHA HORA PROBLEMA SOLUCION TECNICO

Resumen rápido

Columna ticket Puede llamarse TICKET o TKT.
Fila pendiente Debe tener SOLUCION con información y la columna de ticket vacía.
Fila no procesada No será procesada si ya tiene ticket registrado o si falta la solución.
1
Fecha y hora informadas

Si la planilla contiene fecha y hora, el sistema utilizará esos valores durante la carga.

2
Sin fecha ni hora

Si la planilla no contiene fecha ni hora, el sistema utilizará la fecha y hora actuales de la plataforma.

3
Fecha del día actual

Si la fecha corresponde al día actual, la hora no puede superar la hora actual permitida por la plataforma.

Recomendación: lo más recomendable es completar las columnas FECHA y HORA para evitar diferencias durante la creación del ticket.

Recomendaciones y consideraciones importantes

Recomendaciones de uso

  • Utiliza siempre una planilla Excel con formato válido del sistema.
  • Revisa la configuración antes de iniciar la carga, especialmente el usuario reportante y el grupo de trabajo.
  • Verifica que la planilla corresponda al archivo que realmente deseas procesar antes de presionar Enviar.
  • Es recomendable usar la plantilla base del sistema para preparar la información antes de la carga.
  • Si necesitas orientación, revisa los archivos de ayuda incluidos en la aplicación.

Precauciones durante la ejecución

  • Una vez iniciado el proceso, espera a que la página cargue completamente antes de mover la ventana del navegador.
  • Evita interactuar manualmente con la ventana del navegador mientras la automatización se encuentra en ejecución.
  • No cierres la aplicación ni el navegador durante el procesamiento automático.
  • No intentes iniciar una nueva carga mientras exista un proceso en curso.
  • Si ocurre un error, revisa la planilla y vuelve a intentar solo cuando el proceso anterior haya terminado.
Recomendación: al iniciar la carga, espera a que la ventana del navegador termine de abrir y la página cargue completamente antes de moverla o interactuar con ella.
